M-NET API for MWIT

Requirements
------------

-   (Recommended) CentOS 7.x
-   MongoDB 4.x
-   Node LTS

Installation
------------

```bash
yarn
```

Build
-----

For for development

```bash
yarn dev
```

For production build

```bash
yarn build
```

Your complied files will be located at `./build`
